# Motivational site for crypto-investors

## Tech Stack
- HTML
- CSS
- JavaScript

## Description
A bit funny site without real purpose but to practise fetch, async/await, promises, web sockets, and JavaScript overall. 
The site is connected to WhiteBit API through the web socket which provides real time information about price of BitCoin compared to USDT. 
Also on click on "Generate Post" button a random quote will be fetched from https://quotes-db.vercel.app/api and put into the text area below.

## Characteristics
- Use of Web Sockets
- Use of fetches
- Asynchronous code achieved with async/await and use of promises
- Dynamic change of classes in HTML to make site more alive
